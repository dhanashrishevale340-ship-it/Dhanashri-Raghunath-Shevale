# AI Resume Screening & Job Recommendation System

## 📌 Project Overview

The AI Resume Screening & Job Recommendation System is an intelligent web application that analyzes resumes using Artificial Intelligence, Natural Language Processing (NLP), and Machine Learning techniques.

The system automatically extracts candidate details, education, skills, experience, and projects from uploaded resumes. It calculates an ATS (Applicant Tracking System) score, recommends suitable job roles, provides improvement suggestions, maintains resume history, and generates downloadable PDF reports.
